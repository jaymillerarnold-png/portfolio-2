# Font Awesome Pro subset (web)

This folder is a **minimal** copy of a **licensed** Font Awesome **Pro** web kit (CSS + license as shipped).

**Included styles:** classic **regular**, **solid**, and **thin** (plus `fontawesome.min.css` base). The subset in this bundle tracks **Font Awesome 7 Pro**-style class names (see CSS `--fa-family-classic` when present).

**Typical files:**

- `css/fontawesome.min.css`, `regular.min.css`, `solid.min.css`, `thin.min.css`
- `webfonts/*.woff2` as required by those stylesheets (add when refreshing the subset)
- `LICENSE.txt` (Font Awesome license — **keep with any redistribution**)

**Compliance:** Pro assets may not be suitable for **public** hosting. Confirm with legal before pushing webfonts to a public remote.
